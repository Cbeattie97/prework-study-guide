Prework Study Guide

HTML

  • HTML is sharthand for hypertext markup language
  • HTML gives structure to a webpage
  • The head tags encase metadata which contains functional information about the webpage and is unseen on the browser
  • The body tags encase the visible elements elements of the webpage
  • Elements within elements are called "nested elements" they are referred to as parent/child elements
  • "!DOCTYPE" isn't case sensitive

CSS

  • CSS is shorthand for cascading style sheets
  • CSS is the tool used to add style to a webpage
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Coloors are selected either by semantic reference (blue, green etc.) or by hexadecimal code, which is expressed using the #, followed by a 6 digit reference to the desired color
  • Sizes are typically measured in pixels or "px"

Git

  • Git is a version control system
  • Github allows multiple developers to work on a project simultaneously
  • Projects are stored on a repository or "repo"
  • git status; checks what branch we are on
  • git checkout -b branch-name; creates a new branch and switches to it

JavaScript

  • JavaScript adds Logic to our website, allowing users to interact with the page
